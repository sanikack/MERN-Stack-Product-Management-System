# MERN-Stack-Product-Management-System

A full-stack Product Management System built using:
Frontend: React 
Backend: Node.js + Express
Database: MongoDB Atlas
Authentication: JWT
Password Hashing: bcrypt
Styling: SCSS

## USER
Register
Login
View Products

## ADMIN
Create Product
Update Product
Delete Product
View All Products

## BACKEND SETUP
cd backend
npm install
node Server.js

## FRONTEND SETUP
cd frontend
npm install
npm run dev

## ADMIN CREDENTIALS

## SECURITY FEATURES
JWT-based authentication
Role-based access control (Admin & User)
Password hashing using bcrypt
Protected routes (Backend & Frontend)

## NOTES
Admin can access dashboard.
Users can only view products.
Token is stored in localStorage.
